getComputedStyle 获取样式表

window.getComputedStyle​

getComputedStyle方法返回一个只读对象CSSStyleDeclaration对象，该对象包含给定的element的所有CSS属性，这些属性是当前element所应用的最终属性。

各个CSS属性值可以通过对象提供的API，或者使用CSS属性来访问，比如

const style = window.getComputedStyle(document.body);
// 通过css属性直接访问属性值
console.log(style.color) // 'rgb(255, 255, 255)'

// 通过getPropertyValue 传入指定的css属性获取属性值
console.log(style.getPropertyValue('color')) // 'rgb(255, 255, 255)'
提示

有些三方库，比如jquery。调用getComputedStyle方法是从document.defaultView.getComputedStyle获取，实际上和window.getComputedStyle完全相同

console.log(document.defaultView === window) // 输出true

window.getComputedStyle方法的参数​

方法描述
element需要被计算属性的DOM节点
pseudoElt指定要匹配的伪元素，使用这个参数可以指定element的伪类（可选项，默认null, ）

调用方式如下

window.getComputedStyle(document.body);

window.getComputedStyle(document.body, ':after');
提示

建议总是给getComputedStyle方法传入第二个参数, 不需要伪类就传null
因为Firefox3-3.6版本中，第二个参数是必填的CanIUse getComputedStyle

getComputedStyle(element) 与 element.style​

他们都会得到同样的CSSStyleDeclaration对象, 但有一些区别

  1. getComputedStyle 获取的属性是只读的，如果通过属性手动设置会得到报错，如下图


    直接通过 `element.style` 设置css属性可以成功生效，页面文字也变成了红色

  2. [element].style 属性用于给element元素设置样式

  3. [element].style 只能获取到设置在该元素的style属性上的css样式。而getComputedStyle可以获得一个元素最终被应用的所有样式

兼容性​

备注

兼容性参考张鑫旭的文章获取元素CSS值之getComputedStyle方法熟悉

element元素，除了拥有style属性，还可能会拥有currentStyle属性

style 属性返回的是当前元素通过style设置的属性，如果是css样式文件设置的就无法获取到

currentStyle 属性返回当前元素计算后的最终css属性，其返回的结果和getComputedStyle是比较相似的 （该属性ie浏览器支持）

所以在获取一个元素的完整Style计算属性时可以这样兼容

// 如果元素不存在currentStyle属性，则使用getComputedstyle获取
const styles = element.currentStyle || window.getComputedstyle(element, null)

style.getPropertyValue 方法​

getPropertyValue​

方法属于 CSSStyleDeclaration 实例，通过此方法可以获取到实例中的某个CSS属性，比如

const style = window.getComputedStyle(document.body);
console.log(style.getPropertyValue('color')) // 'rgb(255, 255, 255)'

获取元素的CSS变量​

除此以外，getPropertyValue方法还可以获取元素上被设置的css变量，比如：

<style>
  body {
    --color: red;
    color: var(--color);
  }
</style>

<body>
  content
</body>
<script>
  const styles = window.getComputedStyle(document.body);
  console.log(styles.getPropertyValue('--color')) // 最终输出结果 'red'
</script>

设置元素属性​

由于 getComputedStyle 方法获取到的style只能读取属性，所以需要设置style属性时，要使用 element.style, 比如：

// 设置body的color为'red'。第一种
document.body.style.color = 'red';

// 设置body的color为 'red'。第二种
document.body.style.setProperty('color', 'red')

// 设置body的css变量'--color'为'red'
document.body.style.setProperty('--color', 'red')

// 特别注意，不能直接通过style直接设置css变量，下方的设置无效
document.body.style['--color'] = 'red' // 无效

getPropertyValue兼容性​

getPropertyValue方法也有兼容性问题，在IE-8及以下不支持，作为替代，可以使用 style.getAttribute 方法代替

应用​

获取伪元素的属性值​

<style>
  h3::after {
    content: " rocks!";
  }
</style>

<h3>Generated content</h3>

<script>
  const h3 = document.querySelector("h3");
  const result = getComputedStyle(h3, ":after").content;

  console.log("the generated content is: ", result); // returns ' rocks!'
</script>

html2canvas​

html2canvas是一个可以把html元素转化为图片的库，其内部就是使用了getComputedStyle获取dom元素的属性开始绘制canvas

扩展​

CSSStyleDeclaration 对象​

有三种API可以获取到CSSStyleDeclaration对象

  1. 通过 HTMLElement.style 获取，获取的对象里只有当前元素的内联style样式，形如 <div style="...">

  2. 通过 Window.getComputedStyle 获取，获取的对象是只读的。无法更改。

  3. 通过 [CSSStyleSheet](https://developer.mozilla.org/en-US/docs/Web/API/CSSStyleSheet) API获取， 可以通过此路径访问到 document.styleSheets[0].cssRules[0].style
